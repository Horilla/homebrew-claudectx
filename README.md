# homebrew-claudectx

Homebrew tap for [claudectx](https://github.com/Horilla/claudectx) — reduce Claude Code token usage by up to 80%.

## Install

```bash
brew tap Horilla/claudectx
brew install claudectx
```

## Usage

```bash
claudectx analyze        # See where your tokens go
claudectx optimize       # Auto-fix token waste
claudectx report         # 7-day usage analytics
claudectx watch          # Live token dashboard
```

See the [full documentation](https://github.com/Horilla/claudectx#readme) for all commands.

## Update

```bash
brew update
brew upgrade claudectx
```
